Antispam for bbPress 2.0

== Description ==

bbPress Antispam for bbPress 2.0 is inspired on Antispam Bee and working similar.
It is free from Payment and don't send anything to a server.

== Installation ==

1. Download Plugin.
2. Decompress and upload the contents of the archive into /wp-content/plugins/.
3. Activate the Plugin through the 'Plugins' menu in WordPress
